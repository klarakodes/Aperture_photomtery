This IPython Jupyter Notebook conducts aperture photomtery on a selected target in a FITS image. The user can select their own FITS image that will be read by one of the functions which will save its data and header. We then use another function that will loacte all the sources in the image using the Photutils package and the conduct aperture photomtery on a source specified by the user. The position of the target can be inputed by clicking on the target in an interactive window or my inputing pixel or equatorial coordinates (exmaples of this can be found in the Jupyter notebook). The user is also asked to input a calibration constant so the magnitude of the target can be calculated. The function will first show the target with an aperture and annulus in an interactive window. Once the user closes this, the photometry is conducted and the results stored in a Pandas dataframe that is outputed as a CSV file "aperture_photometry_results.csv". See the final note in the Jupyter notebook on how the data is saved. 

I provide a couple of example FITS images that can be used to run this code (see folder "Sample FITS images"). These are images of the LHS 33 region in various photometric filters and have all been taken at and belong to the UCLO observatory.
